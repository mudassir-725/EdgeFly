Quick manual tests (in browser devtools console)

1. Check amadeus test:
EdgeApi.amadeusTest().then(console.log).catch(console.error)

2. Register / Login (then token is stored automatically on successful login):
EdgeApi.register({ email: 'you@example.com', password: 'Secret123' }).then(console.log)
EdgeApi.login({ email: 'you@example.com', password: 'Secret123' }).then(console.log)
// After login:
EdgeApi.getMe().then(console.log)

3. Flight search (guest):
EdgeApi.searchFlightsGuest({
  origin: "LHR",
  destination: "JFK",
  departureDate: "2025-10-25",
  returnDate: "2025-10-31",
  travelClass: "ECONOMY",
  passengers: 1
}).then(console.log)
